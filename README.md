- 2 приложения сервис и апи
- строки подключения в индивидуальных appsetting, не стал делать дополнительный общий
- в сервисе использовал миграцию прямо в проекте, хотя по хорошему выносить в отдельный и делать честные миграции
- graceful shutdown реализовал таймаутом на 5 мин, ибо такое ограничение в задании
- тестировал на 100к записях - среднее время импорта 40 сек. (файл приложил)

- в апи добавил сваггер, открывется по обычному урлу (http://localhost:5205/index.html)
- в апи использовал CQRS, наверняка это избыточно (хотя вся система попадает под этот микросервисный паттерн, в том смысле, что сервис - только запись, апи - только чтение), но просто для демонстрации, что умею/использую.
- в апи использовал AutoMapper для маппинга большого (Office -> OfficeDto) объекта.
